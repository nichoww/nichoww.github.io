Automated Stock Trading Bot

This is a fully automated real-time stock trading bot that connects to the Alpaca brokerage API for paper trading (simulated trading with real market data). The bot monitors 20 [..

Key Workflow:

  1. Fetches live market data every 60 seconds from Alpaca
  2. Generates trading signals using Moving Average (MA) crossover strategy (10-period fast MA vs 30-period slow MA)
  3. Calculates position sizes using ATR (Average True Range) to risk only 2% of portfolio per trade
  4. Places limit orders with automatic fallback to market orders if unfilled after 30 seconds
  5. Manages risk with dynamic trailing stops that ratchet up (never down) as prices rise
  6. Persists portfolio state to JSON so trades survive restarts
  7. Logs all activity to CSV audit trails for analysis
  • Real-time 60-second trading loop
