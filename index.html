Customer Segmentation & Churn Prediction

Engineered an automated data pipeline for customer segmentation and developed a churn prediction system to identify high-risk customer segments.

  • Engineered automated Python/SQL data pipeline analyzing 10,000+ customer records
  • Implemented RFM segmentation methodology for customer categorization
  • Built XGBoost classification model achieving 84% churn prediction accuracy
  • Deployed Flask API serving real-time predictions integrated with Tableau dashboard

How It Works (Demo):

  1. Customer data is extracted from CRM and transaction databases
  2. RFM (Recency, Frequency, Monetary) analysis segments customers into categories
  3. XGBoost model analyzes behavioral patterns to predict churn probability
  4. Flask API receives customer data and returns churn risk scores in real-time
  5. Tableau dashboard visualizes retention metrics and customer lifetime value
  6. Marketing teams receive alerts for high-risk customers with suggested actions

Next Steps:

  1. Integrate with marketing automation platforms for targeted campaigns
  2. Develop personalized retention offers based on churn probability and customer value
  3. Add A/B testing framework to measure effectiveness of retention strategies
  4. Expand model to include customer sentiment analysis from support interactions
  5. Create self-service dashboard for business users to simulate retention scenarios
